# Math Playland

A fun math learning application for kids!

## Features

- **Calculator**: A fully functional calculator with big, easy-to-press buttons.
- **Math Game**: An interactive game to practice Addition, Subtraction, and Multiplication.
  - Three difficulty levels: Easy, Medium, Hard.
  - Score tracking and streak celebrations.

## Technology

This project is built with vanilla **HTML, CSS, and JavaScript**.
It uses [Tailwind CSS](https://tailwindcss.com/) via CDN for styling, maintaining the original design system.

## How to Run

Simply open `index.html` in your web browser. No installation or build steps required!
